# EC463_miniproject
SW mini project for EC463 senior design I

# Prereq 
* Python 3.x
* And install the following libraries _via_ "pip install":
- django
- social-auth-app-django
- geoip2**
- matplotlib**
- postgres**
- django-tz-detect**

# How to Run This App on the Local Server 
```console
$ python manage.py runserver
```
